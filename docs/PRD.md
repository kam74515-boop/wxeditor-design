# WXEditor — 产品需求文档 (PRD)

> **版本**：v2.1 | **更新日期**：2026-04-15 | **产品形态**：Web SaaS

## 1. 产品概述

**产品名称**：WXEditor（微信公众号图文排版编辑器）

**产品定位**：一款面向自媒体创作者和运营人员的专业富文本排版工具，核心场景方面：
1. 提供高质量的微信公众号文章排版体验
2. 支持多人实时协作编辑
3. AI 辅助写作与文案润色（SSE 流式 + Function Calling）
4. 一键同步到微信公众号草稿箱
5. 多公众号管理与定时发布

**竞品参考**：秀米、135 编辑器

## 2. 目标用户

| 用户类型 | 使用场景 |
|----------|----------|
| **自媒体创作者** | 日常撰写微信推文，利用 AI 辅助提升创作效率 |
| **运营团队** | 多人协同编辑、审阅推文，通过团队功能管理权限 |
| **企业新媒体部门** | 批量管理内容，使用模板库保持品牌一致性 |
| **个人博主** | 快速排版并一键发布到公众号 |

## 3. 核心功能模块

### 3.1 富文本编辑器（基于 UEditor）

- **基础排版**：加粗、斜体、列表、标题、引用、表格等完整富文本操作
- **微信 HTML 适配**：
  - **标签白名单过滤**（sanitize-html）：仅保留微信支持的标签（`p`、`span`、`img`、`section` 等）
  - **内联样式转换**（styleConverter）：将 CSS Class 和外部样式转为行内 `style`
  - **微信属性注入**：为图片自动补充 `data-ratio`、`data-w` 等微信原生属性

### 3.2 AI 写作助手

- **智能对话**：基于通义千问大模型（qwen3.5-plus），支持上下文感知的对话式写作
- **SSE 流式输出**：实时打字效果的 AI 响应
- **AI Agent**：Function Calling 工具调用架构，结构化提示词模板
- **文章润色**：优化措辞和表达，保持原意
- **内容操作**：简化/扩展文章内容
- **标题生成**：为文章生成多个有吸引力的标题
- **摘要生成**：自动生成 120 字以内的公众号摘要
- **代码提取**：自动从 AI 回复中提取 HTML 代码块，可一键插入编辑器

### 3.3 微信公众号对接

- **多公众号管理**：绑定/验证/切换多个公众号
- **微信 OAuth 登录**：微信授权登录与账号绑定
- **一键同步**：排版完成后直传公众号草稿箱
- **HTML 复制**：后备方案，一键复制净化后的 HTML 到剪贴板
- **图片代理**：上传图片自动转存微信素材库

### 3.4 多人实时协作

- **WebSocket 实时通信**：基于 Socket.IO 实现毫秒级同步
- **编辑锁控制**：
  - **悲观锁**（默认）：同一时刻仅一人编辑
  - **乐观锁**（可选）：多人同时编辑，版本号自动合并
- **光标共现**：显示其他用户的光标位置与选区
- **即时聊天**：内置聊天面板，边编辑边讨论
- **在线用户**：实时显示协作者列表

### 3.5 模板与素材

- **模板库**：预设排版模板，支持分类筛选与公开/私有
- **素材管理**：图片/视频/音频/文件上传，文件夹管理
- **使用统计**：模板使用次数追踪

### 3.6 团队管理

- **团队创建**：组织级协作空间
- **成员邀请**：邮件/链接邀请机制
- **角色权限**：owner（创建者）、admin（管理员）、member（成员）
- **审计日志**：成员变动记录

### 3.7 用户与会员体系

- **账号体系**：四级权限角色 — `user`、`vip`、`admin`、`superadmin`
- **RBAC 权限**：基于角色的访问控制中间件
- **JWT 认证**：Access Token + Refresh Token 双令牌机制
- **会员套餐**：支持免费/专业/企业等多档套餐
- **结算支付**：支持支付宝、微信支付（预留）

### 3.8 项目管理

- **文档管理**：创建、编辑、归档、删除，支持版本历史
- **文档状态**：draft（草稿）→ published（已发布）→ archived（已归档）
- **评论批注**：树形评论回复、文档内批注
- **多篇文章**：图文消息分组管理（图文合集）

### 3.9 定时发布

- **定时任务**：创建/编辑/取消定时发布任务
- **立即执行**：手动触发立即发布
- **执行日志**：定时任务执行记录与状态追踪
- **调度服务**：后端 SchedulerService 自动调度

### 3.10 管理后台

- **系统概览**：用户统计、收入统计、系统状态
- **用户管理**：用户列表、封禁/解封
- **内容审核**：文档管理、评论管理
- **AI 配置**：多供应商管理、模型参数配置、动态切换
- **公众号管理**：管理端公众号审核
- **商品管理**：会员套餐配置
- **数据统计**：使用量分析
- **系统设置**：全局配置

## 4. 非功能性需求

| 需求 | 说明 |
|------|------|
| **架构分离** | 前后端完全分离（Vue 3 SPA + Express API） |
| **Clean Architecture** | 后端采用 Controller → Service → Repository 分层架构 |
| **MySQL 数据库** | Knex 查询构建器 + mysql2 驱动，支持迁移与种子 |
| **容器部署** | Docker Compose 一键启动（PostgreSQL + 后端 + 前端构建 + Nginx） |
| **开发体验** | `npm run dev` 一键并发启动前后端开发环境 |
| **安全防护** | DOMPurify / sanitize-html XSS 防护，express-rate-limit 限流 |
| **图片处理** | Sharp 图片压缩/裁剪，Multer 本地转存 |
| **API 测试** | Jest + Supertest 自动化测试 |

## 6. 技术栈摘要

|----|------|
| 前端 | Vue 3 + TypeScript + Vite 5 + Pinia + Element Plus |
| 编辑器 | UEditor 定制版 |

| 后端 | Node.js + Express 5（Clean Architecture） |
| 实时通信 | Socket.IO |
| 数据库 | MySQL（Knex + mysql2） |
| 测试 | Jest + Supertest |
| 部署 | Docker Compose + Nginx |
