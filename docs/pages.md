> 本文档定义了 WxEditor 全部页面结构、功能描述、子页面/子状态、以及页面间的交互切换逻辑。
> Pencil 设计稿 (`pencil-new.pen`) 已基于本文档完成全部页面 UI 设计。

## 二、页面详解

### 01 — 登录页

**布局**：灰色背景(`#F2F2F7`) + 居中白色卡片(1000×680, 圆角20, 阴影)，左黄右白双栏

**功能**：
- 左侧（`$brand-yellow` 背景）：品牌展示区（Logo、产品名「WeChat Editor」、Slogan「写出更好的文章」）
- 右侧（白色）：登录表单
  - 标题「欢迎回来」
  - 用户名/手机/邮箱输入
  - 密码输入
  - 「登录」按钮 (`BtnPrimary`)
  - 「没有账号？立即体验」链接 → 跳转 02

**交互出口**：
- 登录成功 → **04 项目管理**
- 点击「注册」 → **02 注册页**

---

### 02 — 注册页

**布局**：灰色背景 + 居中白色卡片(1000×680)，左白右黄（与登录页镜像）

**功能**：
- 左侧（白色）：注册表单
  - 标题「创建账号」
  - 用户名、手机号/邮箱、密码、确认密码、验证码输入
  - 「立即注册」按钮 (`BtnPrimary`)
  - 「已有账号？返回登录」链接 → 跳转 01
- 右侧（`$brand-yellow` 背景）：品牌展示区（Logo、「开启内容创作」）

**交互出口**：
- 注册成功 → **04 项目管理**
- 点击「登录」 → **01 登录页**

---

### 03 — 编辑器（核心页面）

**布局**：三栏布局（左侧面板 260px + 中间编辑区 + 右侧面板 340px），背景 `$layout-bg`

#### 左侧面板 (left-sider)

**顶部**：用户卡片（`$brand-yellow` 背景，圆角 12px）
- 头像、用户名、角色信息

**Tab 区域**（6 个 tab，马卡龙色系）：

| Tab | 颜色变量 | 颜色值 | 功能 | 变体页面 |
|-----|---------|--------|------|---------|
| 模版 | `$tab-blue` | #BAE6FD | 浏览和应用排版模板 | 03a |
| SVG | `$tab-purple` | #DDD6FE | SVG 组件库（搜索 + 双列卡片） | 03b |
| 素材 | `$tab-green` | #A7F3D0 | 图片/媒体素材管理 | 03c（原 03b） |
| 上传 | `$tab-gray` | #E5E7EB | 用户自定义素材上传 | 03d（原 03c） |
| 项目 | `$tab-orange` | #FED7AA | 当前项目文章列表 | 03e（原 03d） |
| 生图 | `$tab-pink` | #FBCFE8 | AI 生成图片工具 | 03f（原 03e） |

> 每个 Tab 已创建独立的编辑器变体页面，展示切换后的面板内容。

**底部**：文章信息区（`$brand-yellow` 背景）
- 文章标题、作者、封面图上传

#### 中间编辑区 (center-editor)

- 格式化工具栏（加粗、斜体、下划线、删除线、字号、颜色、链接、图片、表格、代码）
- 富文本编辑区域
- 底部状态栏（字数统计 + 保存状态）

#### 右侧面板 (right-panel)

- 顶部操作栏（预览、导出、复制代码、发布等）
- AI 助手聊天面板
  - AI 对话历史
  - 输入框 + 发送按钮

#### SVG 动画 Tab (03f 变体)

**新增功能**：SVG 动画效果面板（紫色 `$tab-purple` 背景）
- 搜索栏：搜索动画效果
- 6 个动画效果卡片（半透明玻璃态）：
  - 淡入淡出、旋转、缩放、滑动、弹跳、路径动画
- 每个卡片包含图标 + 动画名称

**交互出口**：
- 无全局导航栏，通过左侧用户卡片退出
- 左侧 Tab 切换不同面板内容

## 四、页面交互流程图

### 认证流程
```
01-登录页 ←→ 02-注册页  (容器均为 1000×680)
    ↓ (登录/注册成功)
04-项目管理
```

### 核心编辑流程
```
04-项目管理 → 03-编辑器 (选择/新建项目)
    ↕
05-模板库 → 03-编辑器 (应用模板)
```

### 编辑器内部 Tab 切换
```
03-编辑器 左侧 Tab (马卡龙色系):
  ├── 模版 Tab (蓝色 $tab-blue)     → 03a 模版列表
  ├── SVG Tab  (紫色 $tab-purple)    → 03b SVG 组件库
  ├── 素材 Tab (绿色 $tab-green)     → 03c 素材网格
  ├── 上传 Tab (灰色 $tab-gray)      → 03d 拖拽上传
  ├── 项目 Tab (橙色 $tab-orange)    → 03e 文章列表
  └── 生图 Tab (粉色 $tab-pink)      → 03f AI 生图
  
  另有 SVG 动画 Tab 变体 (03f-SVG动画)
```

### 账户与付费流程
```
TopNav(专业版/头像) → 09-会员状态
09-会员状态 → 07-定价页 (升级方案)
07-定价页 → 08-结算页 (选择方案)
08-结算页 → 09-会员状态 (支付成功)
```

### 团队管理流程
```
TopNav(团队) → 10-团队列表
10-团队列表 → 11-团队详情 (选择团队)
11-团队详情: 成员列表 / 角色管理 / 团队设置
```

### 管理后台（独立）
```
独立入口 → 12-管理后台
12-管理后台: 仪表盘 / 用户管理 / 内容审核 / AI配置 / 数据统计 / 系统设置
  └── 12a-AI配置: API 供应商管理 + 模型参数配置
```
